\# Sample Mendix App



This MPK contains a sample Mendix app for testing the Isracard Twilio Video Widget.



Features:

\- Room join / leave

\- Camera preview

\- Camera enable / disable

\- Microphone enable / disable

\- Participant connected / disconnected events

\- Focus lost / returned events

\- Error event handling

\- Sample nanoflows for event testing
